Project Summary

This project was an assignment for my coding bootcamp. The purpose of the project was to create a weather app that allows a user to look up the weather in Houston, TX using a weather API.

I took it a step further and added;

  • Geolocation from Google in order to show the weather for the user's current location
  • An input box where the user could look up the location for any city in the world
  • Conversion between Celsius and Fahrenheit
  • Background images that change depending on the weather

Weather App Infomation

This weather app was created using Apixu which I used because they provide a flexible search where you can find cities by their name or zip code. They also provided weather icons and an excel workbook listing all weather conditions.

I also used Google Maps Geolocation API which allows the app to show the user's weather at their current location. If the user blocks their location from the app, they still have the option to type in their city to find out their weather.

The weather is set to fahrenheit by default but the top right corner also gives the user the option to see their weather in either Celsius or Fahrenheit. Choosing Celsius will also change the times to a 24 hour clock.
