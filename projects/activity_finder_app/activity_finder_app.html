Activity Finder App

Project Summary

This project was the first group project created in my bootcamp. This was a front end development project with the below requirements.

  • Use at least 2 remote API's
  • Use github to collaborate with team members

My group decided to create an app to help a user find an activity in a city that they would type in. They would also be able to check what type of activity they are looking for such as indoor, outdoor, sports, shopping, wellness, landmarks, bar, restaurant, and museums.

API Information

This app was created using the following API's;

  • Google Places
  • Google Maps
  • Google Autocomplete
  • Apixu Weather

The Google Places API was used to retrieve information on specific activities in a city. The below code shows how we accessed the API.

4 var activity = "type=";
5 if ($("#check_box1").is(":checked")) {

7 }
8 var fullUrl = "https://maps.googleapis.com/maps/api/place/textsearch/json?query=" + activity + "+in+" + location + "&radius=45000" + apiKey; // The full URL to be used to retrieve the Google Places data

The Google Maps API was used to map the data that was received. Below is an example of the Map for a Houston restaurant search. We also used Google's Marker Clustering which groups any map markers that are close to each other and labels them with the number of places in the area.


The Google Place Autocomplete API was used to show a list of cities as the user types in the input box. We used this together with Google Places to help the user in finding cities and to prevent search errors from the user typing a city name incorrectly.

The below code shows how to access the API to show cities only. The options parameter is optional but leaving it blank would autocomplete for specific places as well as cities.

1 function autoComplete( ) {
2 var input = document.getElementById("location"); // Auto complete input box for locations
3 var options = {
4 types: ["(cities)"] // Limits search results to city
5 };
6 var autocomplete = new google.maps.places.Autocomplete(input, options);
7 }

The Apixu Weather API is used to show the current weather in the right corner of the navigation bar for the city the user is looking for activities in. It was used because it worked well with Google Places Autocomplete once the country was removed from the input field text. Another reason Apixu was used is because they provide weather icons for each weather condition which gives a nice visualization for the user.
